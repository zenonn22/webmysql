Join

Muestra el nombre de cada ciudad junto al nombre del país al que pertenece
cada una.
select City.Name,Country.Name from Country,City where CountryCode=Code;

Muestra el nombre de cada idioma junto al nombre del país en que se habla.
select Language,Country.Name from CountryLanguage,Country where CountryCode=Code ;
Muestra el nombre de las 10 ciudades más pobladas junto con el nombre del
país al que pertenece cada una
select City.Population,City.Name,Country.Name from Country,City where CountryCode=Code order by 1 desc limit 10;
Muestra las ciudades que hay en Ucrania (‘Ukraine’)
select City.Name,Country.Name from Country,City where CountryCode=Code and Country.Name='Ukraine';

Muestra los lenguajes que se hablan en el continente africano ordenados
alfabéticamente.
select CountryLanguage.Language, Country.Name from CountryLanguage,Country where CountryCode=Code and Country.Continent='Africa';

Mostrar la suma de las poblaciones de las ciudades de Rusia (Russian
Federation).
select sum(City.Population), Country.Name from Country,City where CountryCode=Code and Country.Name='Russian Federation';

¿Qué cantidad de la población rusa no vive en una ciudad?
select Country.Population-sum(City.Population), Country.Name from Country,City where CountryCode=Code and Country.Name='Russian Federation';

Join 2

Base de Datos “VideoClub”


● Lista de fechas e importes junto con el nombre y teléfono del cliente asociado.
select Data,Import,Nom,Telefon from FACTURA join CLIENT using (DNI);

● Suma del importe de todas las facturas del día 06/03/2009
select sum(import) from FACTURA where Data='20090306';
● Lista de productos (descripció) facturados en las facturas del cliente con DNI 022222222R.
select Descripcio from DETALLFACTURA join FACTURA using(CodiFactura) where DNI='022222222R';
● Lista los nombres de los clientes y los títulos de las películas que tienen o han tenido en préstamo.
select nom,titol from CLIENT,PRESTEC,PELICULA where CLIENT.DNI=PRESTEC.DNI and PRESTEC.CodiPeli=PELICULA.CodiPeli;  
● Listar el número de películas que tiene o ha tenido en préstamo cada cliente.
select nom,count(*) from CLIENT join PRESTEC using (DNI) group by CLIENT.nom;
● Listar los actores principales de las películas del videoclub. Cada actor sólo puede aparecer una única vez.
select distinct Nom actor from ACTOR join PELICULA using (CodiActor);

● Listar el título de las películas que son del genero de terror.
select titol from PELICULA join GENERE using (CodiGenere) where descripcio='Terror';

● Listar las películas que tienen segunda parte junto al título de esa segunda parte.

select p.titol,pe.titol from PELICULA p join PELICULA pe on pe.CodiPeli=p.SegonaPart;

● Listar películas junto con el número de copias que tiene cada una.

select titol,count(*) from PELICULA join COPIA using (CodiPeli) group by CodiPeli;
● Lista de importes facturados cada día ordenados de forma decreciente.
select data,sum(import) from FACTURA group by data order by import desc;
